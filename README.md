### 安装教程
1. 将主题上传至/usr/themes/文件夹下解压；
2. 后台进行启用；
3. 访问前台查看效果。

#### 使用说明
1. MioMoeV1 PHP版本必须大于或等于7.3 （V2并未做测试）。
2. 本主题适合开发者使用，对于有动手能力的童鞋来说二开如喝水。
